# `start dev`

## Description

Launches workloads for development.

## Usage

```bash
npx workloads start dev
```

## Options

| Option                   | Description                                  |
| -------------------------| -------------------------------------------- |
| `-n, --no-exit`          | Do not exit workloads' containers on close.  |

## Global Options

| Option                   | Description                   |
| -------------------------| ----------------------------- |
| `-h, --help`             | Display help for the command. |
