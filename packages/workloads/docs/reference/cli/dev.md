# `dev`

## Description

Launches workflows.

## Usage

```bash
npx workloads dev start
```

## Options

| Option                   | Description                                  |
| -------------------------| -------------------------------------------- |
| `-n, --no-exit`          | Do not exit workflows' containers on close.  |

## Global Options

| Option                   | Description                   |
| -------------------------| ----------------------------- |
| `-h, --help`             | Display help for the command. |
